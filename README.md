# Task 1: Basic Network Scanning with Nmap

## Internship
Oasis Infobyte – Cybersecurity Internship (1 Month)

## Objective
The objective of this task is to perform a basic network scan using **Nmap** to identify open ports and running services on a target system. This helps in understanding the system’s exposure to potential security risks.

## Tool Used
- Nmap (Network Mapper)

## Target
- Local machine / Virtual Machine (Authorized testing environment)

## Command Used
```bash
nmap -sV <target-ip>
